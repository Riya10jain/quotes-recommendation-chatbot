# 🤖 QuoteBot – AI-Based Quote Recommendation Chatbot

## 📌 Project Overview

QuoteBot is an AI-powered chatbot developed using the Rasa framework.  
The chatbot recommends quotes based on user intent such as:

- Motivation
- Inspiration
- Love
- Humor
- Success

This project was developed as part of the **SkillValet SmartBridge Internship – Epic 3 (Data Collection & Model Building)**.

---

## 🎯 Objective

The objective of this project is to design and train a chatbot capable of understanding user queries and responding with appropriate categorized quotes using Natural Language Understanding (NLU).

---

## 🚀 Key Features

- Intent classification using Rasa NLU
- Multiple user query variations for better accuracy
- Structured training data using YAML
- Defined conversation flows using Stories & Rules
- Interactive chatbot testing via Rasa Shell

---

## 🛠️ Technologies Used

- Python 3.10
- Rasa 3.x Framework
- YAML Configuration
- VS Code
- Git & GitHub
