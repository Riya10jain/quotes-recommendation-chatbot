# Quotes Recommendation Chatbot 🤖

## 📖 Project Overview

The Quotes Recommendation Chatbot is an AI-based chatbot developed using the **Rasa framework**.
The chatbot interacts with users and recommends **motivational quotes** based on user queries.
This project demonstrates the use of **Natural Language Processing (NLP)**, **Natural Language Understanding (NLU)**, and **dialogue management** using Rasa.

---

## 🎯 Problem Statement

People often search for motivational quotes to stay inspired and positive. Manually finding suitable quotes can take time.
This chatbot provides an automated solution that instantly recommends motivational quotes based on the user's request.

## ⚙️ Features

* Responds to greetings from users
* Recommends motivational quotes
* Handles basic conversational queries
* Uses trained Rasa NLU model
* Tested using Rasa Shell

---

## 🚀 How to Run the Project

### 1️⃣ Activate the Virtual Environment

```
rasa_env\Scripts\activate
```

### 2️⃣ Train the Model

```
rasa train
```

### 3️⃣ Run the Chatbot

```
rasa shell
```

## 🧪 Testing

The chatbot was tested using:
